# Presentation Vue.js

Hi there!
Here it is a boilerplate for make a presentation (slides) with your browser using Vue.js

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```

### Run your tests

```
yarn run test
```

### Lints and fixes files

```
yarn run lint
```
